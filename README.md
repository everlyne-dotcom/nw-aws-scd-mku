# AWS Student Community Day Nairobi 2026

A production-ready event website built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- 🎨 Dark theme with glassmorphism design
- ✨ Smooth scroll animations and transitions
- 📱 Fully responsive (mobile-first)
- 🖼️ Hero background slider with zoom effect
- ⚡ Optimized performance
- ♿ Accessible (ARIA labels)

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Customization

- Update event details in `Hero.jsx`
- Modify speakers in `Speakers.jsx`
- Change sponsors in `Sponsors.jsx`
- Adjust ticket pricing in `Tickets.jsx`
- Replace background images in `Hero.jsx`
